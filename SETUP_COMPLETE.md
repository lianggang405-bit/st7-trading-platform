## 🔍 什么是 Schema 缓存？

Supabase REST API 使用缓存机制来提高性能：

1. **缓存内容**：表结构、列信息、关系等
2. **更新周期**：通常 1-2 分钟自动更新
3. **影响范围**：REST API 查询（如应用中的数据库查询）
4. **不影响**：直接 SQL 查询（如您在 Supabase SQL Editor 中看到的）
